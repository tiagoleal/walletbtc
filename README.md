## About

## Getting Started

First, it is necessary to install libs:

```bash
#Install node and libs
npm init -y
npm i bip39 bip32@2.0.6 bitcoinjs-lib --save
#Install electrum

```

Started run the comand
```bash
node ./createWallet.js
```

Generate a wallets
